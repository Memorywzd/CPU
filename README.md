# CPU
Course Design of Principles of Computer Organization  
  
2021年计算机组成原理课设：  
  设计实现一个字长8位的CPU，所设计的系统能调试通过，进行仿真测试后在FPGA开发板上运行一段程序，通过检查程序结果的正确性来判断所设计计算机系统的正确性。
